## üì¶ Setup & Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Msultanularifeen/Sultan-Technologies.git
    cd Sultan-Technologies
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```

4.  **Build for Production**
    ```bash
    npm run build
    ```
